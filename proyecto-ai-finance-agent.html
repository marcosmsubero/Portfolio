Asistente financiero con IA en n8n

Workflow conversacional pensado para consultar gastos, revisar presupuesto e insertar movimientos automáticamente con ayuda de un agente conectado a datos reales.

Qué hace este workflow

Este modelo combina una interfaz conversacional, un agente de IA, memoria persistente y herramientas conectadas para trabajar sobre datos financieros de forma natural. El usuario puede hacer preguntas sobre sus gastos, revisar cuánto presupuesto le queda en una categoría concreta o registrar nuevos movimientos sin interactuar directamente con tablas o bases de datos.

Chat como interfaz

El flujo utiliza conversación natural como punto de entrada para que consultar o registrar datos sea más sencillo.

Memoria conversacional

La memoria permite mantener contexto entre mensajes y hacer el uso del agente más coherente y cómodo.

Datos conectados

El workflow no solo responde: también consulta y escribe sobre las herramientas conectadas.

Qué se puede adaptar

La lógica del workflow puede reutilizarse para otros casos de consulta de datos, registro automático o seguimiento personal.
